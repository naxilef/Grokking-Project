###Grokking in Neural Networks

This project investigates the phenomenon of grokking in neural networks.  
Grokking is when a model initially memorizes the training data (achieving near-zero training loss)  
while test accuracy remains at chance level and then after a long training period, it suddenly generalize and reaches near-perfect test performance. 

This project will reproduces empirical findings from:
*  Modular Addition: https://arxiv.org/abs/2301.02679
*  Grokking Sparse Parity: https://arxiv.org/abs/2303.11873
